## PDF 뷰어 애플리케이션

### 프로젝트 설명

해당 프로젝트는 React.js, Node.js, PDF.js를 활용하여 웹 애플리케이션에서 PDF 파일을 파싱하고, 이를 기반으로 한 PDF 뷰어입니다.

사용자가 PDF 파일을 업로드하면, `pdfjs-dist` 라이브러리를 이용해 텍스트를 이중 배열 형태로 파싱합니다. 이후, 파싱된 데이터를 UI에 표시하여 사용자가 PDF를 쉽게 확인할 수 있도록 합니다.

### 프로젝트 실행 및 설치

```
npm install
or
yarn install
```

```
npm start
or
yarn start
```
